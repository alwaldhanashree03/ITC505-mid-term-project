This webpage utilizes various HTML tags to build the structure:

  • <div>: Used to create different sections like start screen, movie selection, game container, etc.
  • <button>: Buttons are used for user interaction, such as starting the game and selecting options.
  • <h1> and <h2>: Headings are used to display titles and subtitles for the sections.
  • <img>: The <img> tag is used to display images related to the story or game.
  • <p>: Paragraph tags are used to display text-based content such as the story text.
    • <footer>: This section provides the last update information and credits.
